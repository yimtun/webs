伪类选择器 伪类是一种特殊的选择器 伪类选择器定义特殊状态下的样式 这些特殊状态是无法通过 标签 id class 及其他属性实现 来进行精确的控制 鼠标放到链接上会变成小手 点上不松手 变成了 红色 松开鼠标 ################### 链接伪类 链接的四种状态 激活状态 已访问状态 未访问状态 鼠标悬停状态 冒号 加上伪类的名称 :link 未访问的链接 :visited 已访问的链接 :hover 鼠标悬停状态 :active 激活的链接 点击鼠标 松开鼠标之前的状态 ################# 元素和伪类紧挨着 中间没有空格 ################ 清空历史记录进行验证 ########### 伪类 :hover 和 :active 不仅仅用于a标签 也可以用于其他元素 1 hover 用于访问的鼠标经过某个元素时 2 active 用于一个元素被激活时 即 按下鼠标之后 放开鼠标之前的时间 这个两个伪类需要注意兼容性问题 IE6 和更早版本 支持 a 元素的四种状态 IE6 不支持其他元素的 :hover :active ################ 通过伪类给a标签设置状态时 需要 注意顺序 链接伪类的书写顺序 :link :visited :hover :active a:hover 必须位于 a:link a:visited 之后 a:active 必须位于 a:hover 之后 伪类名称对大小写不敏感 ################# 伪类结合 class 类选择器的使用 <a href="http://www.imooc.com" target="_blank" class="one">css使用方法</a> a.one:link{ } a.one:visited{ } a.one:hover{ } a.one:active{ } #################### 伪类选择器 结合后代选择器的使用 <p><a href="http://www.imooc.com" target="_blank">css使用方法</a></p> p a:link{ } p a:visited{ } p a:hover{ } p a:active{ }
