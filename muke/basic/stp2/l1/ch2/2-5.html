css使用方法1 css样式可以写在哪些地方 如何写 四种 1 行内样式(内联样式) 2 内部样式表(嵌入样式) 3 外部样式表 4 导入式 ########################################## CSS 第一种 行内样式 在开始标签内添加style样式属性 如 <p style="color: red">内容</p> ############################################## css使用方法 第二种 内部样式(嵌入样式) 把css样式代码写在 <style type="text/css"> 样式... </style> 说明 style 标签要放在 head 标签之间 针对低版本浏览器的写法 不让样式表的内容显示在页面上 高版本的浏览器会自动忽略注释标签 <style type="text/css"> <!-- p { color: blue; } --> </style> ###############################################
