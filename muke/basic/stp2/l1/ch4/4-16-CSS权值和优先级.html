CSS样式优先级

所谓CSS优先级,指CSS样式在浏览器中被解析的先后顺序

权值相同,就近原则； 权值不同，哪个权值高，使用哪个。
######### 派生选择器的优先级 ######## #### !important 来调整优先级 ####### 可以调整样式规则的优先级 添加在样式规则之后 中间用空格隔开 甚至高于行内样式的优先级 优先级最高 p b { color: yellow !important; } #####CSS优先级总结##### !important 声明最高 CSS使用方法优先级 行内样式》 内部样式》 外部样式 注：link链入外部样式style 内部样式优先级 取决于先后顺序 样式表中优先级 id选择器 》 class选择器 》 标签选择器 》 通配符选择器 权值相同就近原则 权值不同 使用权值高的
